# Eastern Exteriors LLC

Eastern Exteriors LLC is a roofing and siding company. This project covers building their client website and running ongoing SEO for it.

## Claude's Role

Help build the site, then drive it up the SERP for roofing and siding keywords in their service area — and keep it performing once leads start coming in.

If a session is drifting without moving toward a live site ranking on page 1, nudge me back: "Is this moving Eastern Exteriors toward ranking on page 1, or is it a detour?"

## Process

1. **Proposal** — scope and pitch the work to Killah (Owner/CEO)
2. **Site Build** — build the website
3. **SEO Optimization** — optimize for rankings on relevant roofing/siding keywords
4. **Leads Come In** — site starts generating leads
5. **Ongoing Maintenance** — keep the site running and maintain rankings

## Key People

- **Kyle** — Owner/CEO, the client

## Folder Structure

- `00 Proposals/` — scoping docs, pitches, proposal drafts
- `01 Site Build/` — site build work and assets
- `02 SEO Optimization/` — SEO work: keyword research, on-page optimization, technical audits
- `03 Leads/` — lead tracking and notes once the site is live
- `04 Ongoing Maintenance/` — maintenance tasks and logs
- `05 System/` — scripts, config, reusable processes
- `06 Skills/` — skill markdown files (NOT Claude Code skills)
- `07 Attachments/` — images, screenshots, PDFs
- `08 Iteration Logs/` — notes on what to improve

## Rules & Conventions

- **`(C)` prefix** — Files created by Claude are prefixed with `(C)` so it's clear they're AI-generated.
- **Editing rule** — Before editing any file without the `(C)` prefix, ask for permission first.
- **Skills** — All reusable scripts/automations are saved as markdown files in the Skills folder, NOT as Claude Code skills.
